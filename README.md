Running Client:

1. Navigate to frontend folder and run "npm install"
2. Run "npm start"

Running Server:

1. Navigate to backend folder and run "npm install"
2. Run "node index.js"

Run localhost:3000 in the browser window </br>
Server runs on port 1000
